Projetos

Este projeto consistiu no desenvolvimento de uma aplicação que utiliza a API do The Movie Database (TMDb) para oferecer recursos avançados de busca por streaming e pesquisa múltipla. O objetivo era proporcionar aos usuários uma maneira fácil e eficiente de encontrar filmes, séries e informações relacionadas, facilitando a descoberta de conteúdo disponível em diferentes plataformas de streaming. A aplicação foi projetada e implementada com uma interface intuitiva e 100% responsiva, garantindo uma experiência agradável ao usuário.
